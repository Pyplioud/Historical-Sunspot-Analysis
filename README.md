# Historical Sunspot Analysis
Python program which takes the data from the Sunspot Index and Long-term Solar Observations and plots the number of sunspots registered throughout the years.


The generated plot reveals a clear ~11 year cycle in solar activity, characterized by the oscillation between Solar Maximum (high activity) and Solar Minimum (low activity). This phenomenon is known as the Schwabe cycle.
Approximately every 11 years, the Sun's magnetic poles flip, leading to intense magnetic storms that manifest as sunspots, solar flares, and Coronal Mass Ejections (CMEs). Understanding these patterns is crucial for Space Weather forecasting and protecting satellite infrastructure on Earth.
